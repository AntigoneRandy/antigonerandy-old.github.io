给计算机小白的几点小建议

“我真的只会开关机”

事实上，是否是纯小白对于大学毕业后的最终水平并没有太大的影响，除非是拿过牌的大佬，否则大学的成绩还是和自己的学习能力高度挂钩，你该牛逼还是牛逼。

拿OI（信息学竞赛）举个例子，从初中甚至小学就开始学习信息竞赛的同学不在少数，也有人六年OI一场空的；也不乏有从高中才开始接触OI，然后一路高歌猛进拿下IOI（最高级的世界级OI比赛）金牌的。

不过，对于计算机学习，确实有一些好习惯和新手适用的建议应该了解了解

另外，本人在文章中所举得例子是普遍现象，如果在其中发现你干过类似的事情，请不要误会，没有针对谁的意思。

一.好习惯

1. 不使用360，腾讯安全管家之类的国产安全软件

这类安全软件安全防护的本职工作效果很一般不说，而且会强制安装/通过各种隐蔽的方式诱导你安装它的全家桶和其他垃圾软件。而且像显示开机时间，自启“热点新闻”等等乱七八糟花里胡哨的功能会占用大量的系统资源，要不了多久电脑就会卡顿不堪

更有甚者，某安全软件在某些情况下会对编译后生成的exe文件误报毒并将其删除，使得编译运行过程无法正常运行，十分不利于学习计算机

况且咱们是信息安全的学生，只要做到少上不良网站，少下载未知来源的软件（包括部分含有木马的“破解版”软件），被感染的机率是非常小的。现在都0202年了，普通的病毒会被Windows自带的Windows defender拦截住，真正牛逼的病毒360这些也是防不住的，所以赶快把安全软件和它的全家桶卸载掉吧。

如果实在觉得没有个安全软件不放心，也可以试试火绒，国产安全软件中最良心的没有之一，平时很安静也不会捆绑安装，有些诸如联网控制，弹窗拦截等等功能还挺好用的

2. 不把外国的软件（尤其是写代码相关的软件）装在含有中文路径的目录下

这一点解释起来很简单，部分国外的软件不支持中文路径，识别不了，所以会出很多奇奇怪怪的问题。因此，强烈建议把国外软件都装在全英文路径下。（路径的查看方法：找到安装后的文件夹并点击上方的地址栏，如果地址栏中的文字没有中文则说明是正确的）

另外，建议把用户名也改成英文，或者拼音也可以

3. 多多使用搜索引擎

新学习计算机语言的同学通常对于很多专有名词或者一些报错/提示（通常是英语的）一头雾水：

vscode是什么？

mingw是什么？

dl是什么意思？

它提示”Internal Server Error”,是什么意思啊？

它跳了个报错信息出来，是什么意思啊？

我编译的时候跳了一大堆错误出来，可以帮我看看吗？

推荐阅读：《提问的智慧》右键点我链接另存为以下载

“提问时不要总想着让别人授你以鱼，而是授你以渔，知道在哪里找到问题的答案有时比问题的答案本身更重要” ——vv

出现问题或者需要某种方法时，建议先使用搜索引擎（如百度，谷歌）搜索一下，你遇到的大多数问题前辈也遇到过，很多问题都可以用搜索引擎得到答案（尤其是关于各种你看不懂的专有名词）

对于报错信息其实不用太过害怕，初学时可能只是因为一个很小的错误（比如忘了打分号）而引起连锁反应,产生一大堆错误信息，甚至有时还会跳转到头文件。但其实不用怕，从最早的错误信息开始读起，把英语知识用起来，不会的词直接上网查。这样不仅锻炼了英语能力，下一次遇到一样的问题也可以一眼看出。用不了多久就会学会的，千万不要一看到报错就头大害怕。

如果实在是无法解决的问题要询问他人，这里提供几个问问题时候的好习惯：

1. 请不要用手机拍电脑屏幕

在非必要情况下（比如电脑是新的），请不要使用手机拍屏幕的方式来向他人展示你的屏幕，这样会造成角度差，图片模糊，抖动等许多问题，严重影响阅读体验。请使用截图程序来辅助你捕获屏幕上的内容（以下是对比）

通过对比，孰高孰低应该非常明了了。

ps: 关于截图工具：

a. 微软赐予你的截图键：可以使用Windows 10自带的截图功能（快捷键Win+Shift+S，框选你想截图的画面后点击右下角弹出的窗口即可打开截图，可以复制后通过通讯软件发送）

b. 马化腾赐予你的截图键： 在电脑QQ运行时，在任何时间同时按下Ctrl+Alt+A即可截图，截图完成后的结果自动保存在剪切板中（一些版本的qq会直接打开窗口）

c. 其他单独的截图工具： 推荐本人最爱的Snipaste，具有简易修图，抓取RGB色号，像素位精确截图，截图贴图等等丰富便捷的功能，下载安装和使用方法请自行百度搜索，此处不再赘述

2. 请不要直接复制代码并粘贴在QQ中发送

这也是小白常踩的雷区，调试代码时往往对方可能会要求“代码发我看看”，但是如果直接把代码复制粘贴过去，不仅长长的一坨会把之前的消息记录挤开，而且宋体且没有高亮的代码真的是丑到极致，让人完全没有想看的欲望，有时还会影响缩进和排版，浪费很多的时间。

对此，推荐两种方法解决：

  1. 使用Ubuntu pastebin

Ubuntu pastebin是一个免费开源的网站，它专门用于粘贴代码并提供高亮，然后生成一个包含代码的网址。你只需要将你的代码粘贴进去然后选择对应的语言，输入作者后点击Paste！就会生成一个包含你代码的网页。你只需要把这个网址发送出去就可以了，其他人点击这个网址进入后就可以自由查看或复制你的代码。

  1. 使用截图工具截下编辑器中代码后发送出去

3. 请懂得感恩，如果不是很熟悉的同学或者找学长/学姐咨询解决了问题，可以发红包致谢

别人的时间也是时间，大家的时间同等重要。如果你是问问题的人，要明白其他人和你没有直接的利益关系，没有人天生就该是你的工具人。帮你是情分，不帮你才是本分。如果别人耐心解答了你的问题，请一定一定要向别人表达感谢，永远不要take it for granted. 如果是不熟悉的人，可以用红包/请吃饭的方式致谢，这样也有利于拉近关系。如果别人上次帮了你下次没帮你，也不要产生怨恨之情，先从自己身上找找原因。凡事先多想想自己配不配。

如果你是那种被各种问题缠的喘不过气，不想当工具人但又不好意思直接拒绝对方的“终身免费顾问”， 可以了解一下“期望值管理”的心理学技巧，这或许可以解决你的烦恼。

此外，适当得接受下别人的小心意也是一件好事，不过要分清对方是真的想感谢你还是 假巴意思

4. 下载软件时候请去官网，拒绝“xx软件园”，“xx资源网”

事实上，绝大多数软件的官网都是下载其最可靠的方式，没有之一（靠谱的博客或者个人网站也可以）。但是由于某黑心搜索引擎搞竞价排名恰烂钱，有很多诸如xxx软件园，xxx资源网的流氓网站就排在了搜索结果的前几位，而且部分软件的官网只有洋文界面，这会使人望而却步。流氓资源网站的套路在于进入过后会看到一个大大的“高速下载”，旁边在下面批注一句很小的“使用高速下载器”。点击这个按钮下载下来的东西不是你要的软件，而是它所谓的“高速下载器”。

“高速下载器”不仅不能把下载速度加快，还会在安装时用尽套路安装恶心人的捆绑软件，他们通过推广其他流氓软件的方式来盈利，流氓又会再安装流氓，一传十十传百，不久就会让你的电脑被流氓软件占领，大大拖慢系统运行效率和使用体验。

如果非要在这上面下载，请费心找找“普通下载”，可能下载下来的东西会阳间一点。

二. 膜拜和装弱的风气以及一些常见黑话科普

在计算机圈内普遍有膜人/装弱等等的风气（主要是oi圈）。圈内人一般称自己“好菜啊”“好弱啊”“好辣鸡啊”，用尽各种谦辞来贬低自己；用“您”或者“您们”（实际上“您们”这个用法是不符合中文语法的）称呼其他人，称对方“大佬”“神仙”“太强啦”，用尽各种敬辞来称呼他人，这种称呼方法能表现出对别人的尊敬和自己的谦虚，但也有一种滑稽的意味。

这种风气可能起源于清华的“贵系”，详细信息可以看下面的图片：

现在来说，个人认为膜拜的一般含义在于好朋友之间互相恶心或者不熟的同学之间以礼相待等等，如果你问问题时采用谦辞，会给别人留下有家教的好印象。有时候别人做出来一道难题时在群里炫耀炫耀，你也完全可以复读一波xxxtql满足下他人的自尊和虚荣心，增进下感情。但有时过度的谦虚或者过度的膜拜可能被认为是在阴阳怪气，反而招人反感。因此请做一个有智商的同时有情商的人，把握好这之间的一个平衡和限度。个人比较讨厌阴阳怪气（开玩笑的除外），有什么意见建议直接说，不要采取阴阳怪气的方式恶心他人，这种方式十分招致反感。

此外，如果你不喜欢这种商业互吹的风气，可以直接朝带头的人私聊提出意见，相信心善的同学们是会接受的。不过适当接受一下其实可以拉近同学间关系。

最后，这种方式有较大的揶揄意味，你可以认为其他人是在开玩笑，不要太把别人的话当真了，也不要在别人正经讲话/发言/问问题的时候冒一句xxxnb,xxxtql出来，否则可能会把正常的消息挤下去，请根据消息的重要程度谨慎斟酌是否膜/复读。

最后科普一些常见的黑话，常常使用由首字母缩写组成的话加上！等标点符号，并且经常有复读的现象出现。如果不懂这些首字母或者黑话代表的含义，盲目复读可能适得其反。（比如别人和很熟的同学开玩笑说xxx（同学名字首字母）lj（辣鸡的首字母），你也跟着复读一句xxxlj,如果恰好那位同学和你不熟且比较敏感，可能你俩就结下梁子了。

常见黑话和缩写全名科普（有一些也是网络流行的缩写，这里一并列出）：

前三/两个字母一般是姓名首字母拼音的缩写，有时也取外号/名字内谐音等等

dalao=dl=大佬

sdl=是大佬

ls=老师（通常系内会有一些杂课或者比较难的课是同学讲的，这种时候可以称其为xxxls,或
者直接取姓的首字母xls）

julao=jl=巨佬（少见，一般直接打中文）

tql=太强了，最常见的夸人用词

tql,wsl=太强了，我死了，因为首字母和“天气冷，我睡了”相同，所以有类似的表情包出现（把缩写故意错打成其他的汉字短语，也有一种幽默的感觉在里面）

gg=哥哥（敬辞）

dd=弟弟（用于自称）

ddw=带带我（用于求大佬带，有时和上一条联动成dldddd（大佬带带弟弟））

nb=牛逼，也是常见夸人的词，但在卢本伟牛逼!（lbwnb！）普及后似乎不太常用了

yy=爷爷，有时候用在名字最后一个字缩写为y的人身上，是比较高等的尊称，但有时也有一丝滑稽的意味

yyds=永远的神，来自“乌兹，永远滴神！”

fw=five=废物（谐音），常见自称，也有好朋友开玩笑的

yysy=u1s1=有一说一，表“就事论事”的意思

qs=确实，表示肯定他人

lj=辣鸡=垃圾，同上

ddw=带带我

tcl=太惨了=太草了=太菜了，这三个意思不尽相同，”太惨了”就是字面意思,用来同情别人;”太草了”一般指很有槽点，很搞笑（单个“草”也是这个意思（不是骂人的），来源于日语“生草”，常见用法还有“过于生草”），“太菜了”就是在卖弱，一般都用wtcl（我太菜了）

T1/T2=第一题/第二题，念“替1替2”，竞赛常见的称呼方法，d1t1一般指的是第一天的第一题，其中d是day的意思

gkd=搞快点，通常用于催别人 发色图

smdh=什么都会，也是常见的夸人用词

smg=什么鬼

另外，遇到自己不认识的专有名词也可以百度解决，网络流行语可以试试小鸡词典

希望大家能牢记这些方便别人也方便自己的习惯，加油~
